# Hider Metaball Body Prototype

`SM_HiderMetaball_Body.obj` is a paintable body-shape prototype for the hider character.

- Height: about 176 cm
- Style: smooth metaball-like T-pose with separated limb volumes
- Material slot: `PaintableBody`
- Intended use: broad, simple surfaces for camouflage painting tests

The mesh is generated by `generate-hider-metaball-body.mjs`. Adjust the blob centers,
radii, strengths, `threshold`, or `scaleToCm`, then rerun:

```powershell
node .\HideSeek\SourceAssets\Characters\Hider\generate-hider-metaball-body.mjs
```

For Unreal import, bring in `SM_HiderMetaball_Body.obj`, create a dynamic material for
the `PaintableBody` slot, and apply the camouflage color/texture workflow there.
